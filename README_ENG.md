\# 🚀 NgKyan's Development Workspace



\*\*Welcome to Nguyễn Kỳ An's source code repository.\*\*



This is a space for experimentation and development of advanced data structures, high-performance libraries, and modern programming solutions. With many years of experience in systems programming and application development, I focus on building tools that help developers write code faster, run more efficiently, and maintain more easily.



\## ✨ Design Philosophy



Every line of code in this workspace is built upon \*\*three core pillars\*\*:



1\. \*\*Performance First\*\*  

&#x20;  Leveraging the full power of the language (C++17, C#) to achieve the highest processing speed with the lowest resource consumption. The libraries are deeply optimized for memory usage, cache efficiency, and runtime performance.



2\. \*\*Developer Experience\*\*  

&#x20;  Code is not just for machines to run — it must feel pleasant for human readers. I prioritize concise, safe, and modern syntax that feels natural (Python-style indexing, LINQ-like operations, etc.).



3\. \*\*Keep It Simple\*\*  

&#x20;  Designed with modularity in mind — header-only in C++ and idiomatic .NET in C# — making integration into other projects fast and hassle-free.



\## 🌟 Featured Projects



\### 📁 C++ Workspace – Optimized Store Library for C++17

\- \*\*store.hpp\*\*: A fully optimized generic container `store<T>` for C++17.

\- \*\*Performance gains\*\*: 25-40% improvement on typical workloads (string joining 51% faster, searching 22% faster, cloning 10% faster…).

\- Makes full use of modern C++17 features: `std::string\_view`, `std::optional`, structured bindings, `constexpr`, `\[\[nodiscard]]`, move semantics, and more.

\- Cleaner code, stronger type safety, better maintainability, and high backward compatibility.



\### 📁 C# \& .NET Workspace – Store<T> for Games \& OOP Applications

\- \*\*Store<T>\*\*: A complete C# version converted from the C++ implementation, written in idiomatic .NET style.

\- Full support for: Python-style negative indexing, slicing, clone system, thread-safe error management (`ThreadLocal`), LINQ integration, collection initializers, and more.

\- Seamlessly integrates with Unity, WPF, MAUI, and other .NET 10+ projects.

\- Currently under active development to better support game development and enterprise applications.
